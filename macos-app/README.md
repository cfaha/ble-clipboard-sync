# macOS App Template (BLE Clipboard Sync)

This folder contains the Xcode template project to build the macOS app wrapper.

- Project: `template.xcodeproj`
- Source: `template/`
- Core logic file to integrate: `../mac/ClipboardSyncMac/ClipboardSyncMac.swift`

## Next steps
1. Open `template.xcodeproj` in Xcode.
2. Add `ClipboardSyncMac.swift` to the project target.
3. Ensure AppKit menu bar integration and BLE permissions in Info.plist.
4. Build & run.
